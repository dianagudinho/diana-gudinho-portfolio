## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Portfolio Sections

- **Home**: Hero section with animated background and key expertise areas
- **About**: Professional summary and technical skills showcase
- **Experience**: Timeline of work experience including Research Assistant at UB, Graduate Student Assistant, Data Analyst at TCS, and Software Engineering Intern
- **Education**: MS in Data Science from University at Buffalo and BE in Information Science from VTU
- **Projects**: Showcase of AI/ML projects including RAG systems, LLM applications, and forecasting models
- **Contact**: Contact information and message form

## Technologies Highlighted

- **Programming**: Python, SQL, R, Scala
- **ML & AI**: PyTorch, TensorFlow, Keras, LLMs, RAG, Multi-Agent Systems
- **Cloud & Tools**: AWS (S3, Glue, Redshift, SageMaker), GCP BigQuery, Airflow, dbt
- **Visualization**: Tableau, Power BI, Matplotlib, Seaborn

### Color Theme

The color theme can be customized in `tailwind.config.ts`. The current theme uses:
- Primary: Blue tones (primary-*)
- Accent: Purple tones (accent-*)
- Dark: Dark gray/blue tones (dark-*)
